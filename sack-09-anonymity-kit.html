Ghost Stack

Infrastructure for creating without a face. By 2030, anonymity is a service.

The Foundation

Basic separation between you and your work.

  • Dedicated device for ghost work (no personal accounts)
  • Tailscale network for encrypted mesh access
  • ProtonMail or Tutanota for project communication
  • Separate payment rails (privacy.com, crypto)
The Middle Layer

Systems that make ghost creation scalable.

  • Self-hosted Bop Gun instance (your tools, your control)
  • Milvus vector DB for memory without attribution
  • Resident artist pipeline (automated, faceless)
  • Content escrow (release without source trace)

The Vanishing Point

Complete separation. Untraceable creation.

  • Decentralized hosting (IPFS, Arweave)
  • Multi-hop payment routing
  • Attribution decay (automatic metadata stripping)
  • Dead man's switch (work releases if you disappear)
